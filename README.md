## Setup Instructions

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Add Your Character Image**:
   - Place your PNG image in the `assets/` folder
   - Name it `character.png`
   - Recommended size: 300x400px or larger

3. **Run Development Server**:
   ```bash
   npm run dev
   ```

4. **Build for Production**:
   ```bash
   npm run build
   ```

## Theme System

The portfolio features 6 different color themes that automatically rotate every 10 seconds:

1. **Purple** - Professional and creative
2. **Blue** - Trustworthy and calm
3. **Green** - Growth and success
4. **Red** - Energy and passion
5. **Orange** - Innovation and enthusiasm
6. **Teal** - Balance and sophistication

## Customization

- **Colors**: Modify the CSS variables in `style.css`
- **Content**: Update the HTML in `index.html`
- **Functionality**: Enhance the JavaScript in `main.js`
- **Themes**: Add or modify themes in the `themes` array
